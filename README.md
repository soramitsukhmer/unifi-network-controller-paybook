# UniFi Network Controller for Soramitsu Khmer

Install and Deploy UniFi Network Controller for Soramitsu Khmer

**Requirements**:
- Ansible AWX or Ansible 2.10.3
- Server: Ubuntu 20.04
- CPU: 2 Cores
- RAM: 4 GB

**Usage**:
```sh
ansible-playbook -i inventory.ini playbooks/provision.yml
```
